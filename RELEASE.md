# GLM-OCR v1.0.0

基于 GLM-OCR 模型的本地图像文字识别工具。

## 功能

- **文本识别** - 通用文字提取
- **文档解析** - 结构化文档识别
- **表格识别** - 表格内容提取
- **公式识别** - 数学公式识别
- **二维码** - 生成与识别
- **批量处理** - 多文件/文件夹批量识别
- **剪贴板识别** - 直接粘贴截图识别

## 系统要求

- Windows 10/11 64位
- 显存 4GB+ (推荐 NVIDIA GPU)
- 内存 8GB+
- 磁盘空间 4GB+

## 下载说明

| 文件 | 说明 |
|------|------|
| `GLM-OCR-Portable.zip` | 便携版（推荐），解压即用 |
| `GLM-OCR-Setup.exe` | 安装版 |

### 便携版使用

1. 下载 `GLM-OCR-Portable.zip`
2. 解压到任意目录
3. 双击 `start.bat` 启动

### 安装版使用

1. 下载 `GLM-OCR-Setup.exe`
2. 运行安装程序
3. 从开始菜单或桌面启动

## 模型说明

本程序使用 [zai-org/GLM-OCR](https://huggingface.co/zai-org/GLM-OCR) 模型。

- 模型已内置于安装包中
- 完全离线运行，无需联网
- 支持 GPU 加速（自动检测）

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Q` | 快速识别 |
| `Ctrl+V` | 剪贴板识别 |
| `Ctrl+O` | 选择图片 |
